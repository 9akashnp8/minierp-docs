# Setting Up Your Employee App

Setup the prerequisites for working with the Employee App.

<Callout type="info">
    - Departments, Designations & Locations/Branches are managed via the Admin Panel
    - The [_master-admin_](/inside/admin-user) user or users with the _hardware-admin_ and _employee-admin_ [role/group](/inside/groups) have access to Admin Panel.
    - Employee ID Prefix Customization is done on the front end by a _hardware-admin_ or _employee-admin_ group user.
</Callout>

## 1. Add Departments
> Departments are divisions within a business that specialize in contributing to the overall functionality of the business.

![Video How to Add Departments](https://i.imgur.com/Qz1MxNn.gif)

## 2. Add Designations
> A designation is a title, description or an official name that a person holds in a company or a place.

![Video of How to Add Designations](https://i.imgur.com/bwAPnT7.gif)

## 3. Add Locations/Branches
> Location/Branches are the different offices locations the company operates from.

![Video of How to Add Locations/Branches](https://i.imgur.com/YikkJO7.gif)

## 4. Customise Employee ID Prefix
> It is the Prefix of the unique Employee ID that all employees are assigned.
